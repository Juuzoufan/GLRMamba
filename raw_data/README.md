# Raw Data Directory

Place the PEMSD datasets here in LibCity format.

Expected layout:

```text
raw_data/
  PEMSD3/
    PEMSD3.geo
    PEMSD3.rel
    PEMSD3.dyna
    config.json
  PEMSD4/
    PEMSD4.geo
    PEMSD4.rel
    PEMSD4.dyna
    config.json
  PEMSD7/
    PEMSD7.geo
    PEMSD7.rel
    PEMSD7.dyna
    config.json
  PEMSD8/
    PEMSD8.geo
    PEMSD8.rel
    PEMSD8.dyna
    config.json
```

Dataset files are intentionally not committed. See `../DATA_PREPARATION.md`.
